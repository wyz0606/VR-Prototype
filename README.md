# VR/MR-Prototype

---

## About the Project

This project aims to show my expertise in Unity Editor Tool building, OpenXR Extensions usage, UI development and Tech Art. It includes 3 main features working on Meta Quest 3. Currently under development targeting to be finished before April 3rd.

## Features

Highlight the core functionalities of your application:
* **Feature 1:** Save and load Spatial data from your headset to Unity scene, allowing you to test without putting on the headset. (Complete)
* **Feature 2:** Ping pong with the wall. You have choices of brackets and balls to play ping pong against your physical wall.
* **Feature 3:** Pinball shooter. You can paint your physical room in the headset with pinball guns.

---

## Getting Started

Unity Version: 6.3.11f1  
Please start your scene from Assets/Project/Scenes/00-Main

### Testing

You may need to use Meta Horizon Link or make a build to test this prototype on your Meta Quest 2 or 3.
